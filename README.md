# GhostWatch — Ghosting Risk Predictor

**Project:** Predictive Modeling of Sudden Communication Cessation in Romantic Relationships  
**Module:** Final Year Project (Computer Science / Computer Engineering)

---

## Files

| File | Purpose |
|---|---|
| `ghosting_prediction_dataset.csv` | Theoretically grounded synthetic training dataset (2,000 rows, 24 features) |
| `dataset_codebook.md` | Full feature descriptions and source citations |
| `dataset_builder.py` | Python script that generated the dataset (include as appendix) |
| `model_pipeline.py` | Full ML training pipeline — runs all 4 classifiers, selects best, saves artefacts |
| `ghosting_model.pkl` | Trained model (generated by running `model_pipeline.py`) |
| `model_comparison.csv` | Cross-validated performance of all 4 classifiers |
| `feature_importance.csv` | SHAP feature importance values |
| `training_report.txt` | Human-readable training summary |
| `app.py` | Streamlit inference interface |
| `requirements.txt` | Python dependencies |

---

## Setup & Run

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Train the model (generates ghosting_model.pkl)
python model_pipeline.py

# 3. Launch the interface
streamlit run app.py
```

## Deploy to Streamlit Community Cloud (free)

1. Push all files to a GitHub repository
2. Go to [share.streamlit.io](https://share.streamlit.io) → New app
3. Connect your GitHub repo, set Main file to `app.py`
4. Click Deploy — live in ~2 minutes
